# CSV

This directory contains structured datasets used throughout the repository.

The CSV files preserve aggregated observations, intermediate datasets, and tabular research results that support the corresponding Markdown articles and PDF documents.

Readers are not expected to browse this directory as a starting point.

Instead, CSV datasets should normally be accessed through the corresponding Markdown articles or supporting research documents.

---

## Repository Relationship

```text
README.md
    ↓
ROADMAP.md
    ↓
articles/README.md
    ↓
Markdown Articles
    ↓
PDF Documents
    ↓
CSV Datasets
```

Markdown articles provide conceptual summaries and repository navigation.

PDF documents provide detailed observations and supporting discussion.

CSV datasets preserve structured research data for reproducibility and future analysis.

---

## Repository Philosophy

Markdown articles, PDF documents, and CSV datasets serve different purposes.

Markdown articles function as:

* Navigation
* Conceptual overview
* Knowledge graph nodes
* Entry points

PDF documents function as:

* Detailed observations
* Validation results
* Methodology
* Supporting evidence
* Long-form discussion

CSV datasets function as:

* Structured observations
* Aggregated datasets
* Statistical source data
* Reproducibility support
* Future analysis resources

The three formats are intended to complement each other rather than duplicate the same content.

---

## Navigation Philosophy

CSV datasets are intended to support reproducibility and future analysis rather than serve as standalone reader-facing documents.

Readers should normally use the corresponding Markdown articles and PDF documents to understand the context and interpretation of the data preserved in this directory.
