# Images

This directory contains the image assets used throughout the repository.

The images are organized as supporting components of the repository's knowledge graph rather than as standalone illustrations.

Many figures represent conceptual models, validation results, or navigation structures derived from observation-based research.

Readers are not expected to browse this directory directly.

Instead, images should normally be accessed through the corresponding Markdown articles.

---

## Figure Categories

The repository currently contains several types of figures, including:

* Representative Figures
* Conceptual Models
* Validation Results
* Repository Navigation Figures
* Infographics

Each figure is intended to support the corresponding article rather than replace it.

---

## Repository Relationship

```text
README.md
    ↓
ROADMAP.md
    ↓
articles/README.md
    ↓
Markdown Articles
    ↓
Images
```

Articles provide the conceptual context.

Images provide visual support for those concepts.

---

## Naming Convention

Localized figures may include language suffixes such as:

* `-en`
* `-jp`

to distinguish language versions.

Where possible, related figures share a common conceptual base name across languages.

---

## Navigation Philosophy

Images are intended to be accessed through their corresponding articles rather than by filename or directory structure.

The Markdown articles provide the necessary context for interpreting each figure.

Without that context, many images intentionally omit detailed explanations in order to remain concise and reusable.
