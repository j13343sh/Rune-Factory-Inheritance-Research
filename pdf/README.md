# PDF

This directory contains the printable research documents used throughout the repository.

The PDF documents provide detailed observations, validation results, methodology, and long-form discussions that complement the corresponding Markdown articles.

Readers are not expected to browse this directory as a starting point.

Instead, PDF documents should normally be accessed through the corresponding Markdown articles.

---

## Repository Relationship

```text
README.md
    ↓
ROADMAP.md
    ↓
articles/README.md
    ↓
Markdown Articles
    ↓
PDF Documents
```

Markdown articles provide conceptual summaries and navigation.

PDF documents provide detailed research materials and supporting evidence.

---

## Repository Philosophy

Markdown articles and PDF documents serve different purposes.

Markdown articles function as:

* Navigation
* Conceptual overview
* Knowledge graph nodes
* Entry points

PDF documents function as:

* Detailed observations
* Validation results
* Methodology
* Experimental discussion
* Long-form reference materials

The two formats are intended to complement each other rather than duplicate the same content.

---

## Navigation Philosophy

PDF documents are intended to be accessed through their corresponding articles rather than by filename or directory structure.

The Markdown articles provide the necessary context for understanding the research documents contained in this directory.
