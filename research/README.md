# Research

This directory contains the supporting research materials used throughout the repository.

The files in this directory preserve validation processes, raw observations, methodology, and experimental records that support the corresponding Markdown articles and PDF documents.

Readers are not expected to browse this directory as a starting point.

Instead, research materials should normally be accessed through the corresponding Markdown articles or PDF documents.

---

## Repository Relationship

```text
README.md
    ↓
ROADMAP.md
    ↓
articles/README.md
    ↓
Markdown Articles
    ↓
PDF Documents
    ↓
Research Materials
```

Markdown articles provide conceptual summaries and repository navigation.

PDF documents provide detailed research discussions and supporting evidence.

Research materials preserve the underlying validation process and experimental methodology.

---

## Repository Philosophy

Markdown articles, PDF documents, and research materials serve different purposes.

Markdown articles function as:

* Navigation
* Conceptual overview
* Knowledge graph nodes
* Entry points

PDF documents function as:

* Detailed observations
* Validation results
* Methodology
* Experimental discussion
* Long-form reference materials

Research materials function as:

* Raw observations
* Validation records
* Experimental methodology
* Supporting datasets
* Research archive

The three formats are intended to complement each other rather than duplicate the same content.

---

## Navigation Philosophy

Research materials are intended to be accessed through their corresponding Markdown articles or PDF documents rather than by filename or directory structure.

The higher-level documents provide the necessary context for interpreting the validation records and experimental materials contained in this directory.
